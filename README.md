- 한줄 소개 : 용도 잡고 근육도 얻고 1석 2조
- 서비스 명 : **`GYM & GLORY`**


### 타겟 🎯

- 운동하고 싶은데, 헬스장 갈 돈이 없는 사람들

- 운동하고 싶은데, 혼자 하기 싫은 사람들
- 운동하고 싶은데, 게임도 하고 싶은 사람들
